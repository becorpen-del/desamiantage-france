Politique de confidentialité

Dernière mise à jour : 2024

Desamiant'Pro collecte vos données afin de traiter vos demandes de devis et de vous mettre en relation avec un partenaire certifié.

Données collectées

  • Nom, prénom
  • Coordonnées (e-mail, téléphone, ville, code postal)
  • Description du projet
  • Paramètres marketing (utm, gclid)

Durée de conservation

24 mois maximum après le dernier échange.

Vos droits

Vous pouvez demander l'accès, la rectification, l'opposition ou la suppression de vos données.
